What is PayStabl?​

PayStabl enables AI agents to handle payments autonomously through programmable smart wallets. Built on Coinbase's developer platform, it provides:

  • x402 Protocol Support - Automatic handling of HTTP 402 Payment Required responses
  • Agent-to-Agent Payments - Direct compensation between AI systems
  • Policy Governance - Spending limits, allowlists, and session controls
  • Enterprise Security - Gasless transactions and audit trails

Getting Started​

# Install the PayStabl SDK
npm install @paystabl/sdk

# Initialize your agent
import { PayStablAgent } from '@paystabl/sdk';

const agent = new PayStablAgent({
  apiKey: process.env.PAYSTABL_API_KEY,
  wallet: { privateKey: process.env.AGENT_WALLET_KEY }
});

Example: Autonomous API Payment​

// Agent automatically pays for premium data
const response = await agent.pay_api_endpoint({
  url: "https://api.weatherservice.com/premium",
  agentId: "weather_agent_001"
});

// Use the payment header in your request
const data = await fetch(response.url, {
  headers: { "X-Payment": response.xPaymentHeader }
});
